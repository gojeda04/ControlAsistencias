¿Qué es HTML?

Todo el mundo usa la web, pero muy pocas veces nos detenemos a pensar como está construida. El HTML (Lenguaje de marcado de hipertexto) es el más básico componente de desarrollo de la Web. Define el significado y la estructura del contenido web.

En la actualidad tiene como objetivo indicar como va ordenado el contenido de una página web. Esto lo hace por medio de las marcas de hipertexto las cuales son etiquetas conocidas en inglés como tags.

Tags principales

Además de estructurar el contenido de la web, también las etiquetas nos facilitan subrayar textos, usar negritas, añadir imágenes, definir hipervínculos entre muchas más funciones. A continuación se muestra una lista de las etiquetas más importantes:

  • < html >: indica el comienzo del documento HTML.
  • < head >: indica que empieza la cabecera de la página. En ella se suele poner el título < title >
  • < body >: es el cuerpo de la página, donde va lo que se ve en el navegador.
  • < h1 >, < h2 >, etc.: son los títulos o encabezados.
  • < a >: define los enlaces.
  • < table >: es una tabla, y dentro de esta tenemos filas < tr > y celdas < td >.
  • < p >: el texto dentro de esta etiqueta forma un párrafo.
  • < img >: imágenes.
  • < ul >: los textos dentro de esta etiqueta se estructuran en listas. Mediante el uso de < li > definimos cada elemento de la lista, y usando < ol > tendremos listas ordenadas.
  • < b > y < strong >: se utilizan para resaltar el texto.
  • < u >: texto subrayado.
  • < i > y < em >: texto en cursiva.

Semántica de HMTL5

La especificación HTML5 trae muchos nuevos elementos a los desarrolladores web, permitiéndoles describir la estructura de un documento web con semántica estandarizada.

Todo el contenido incluido dentro del elemento < body > es parte de una sección. Las secciones en HTML5 pueden ser anidadas. Además de la sección principal, definida por el elemento < body >, los límites de la sección son definidos explícita o implícitamente. La secciones definidas explícitamente son el contenido definido en las etiquetas < body >, < section >, < article >, < aside >, < footer >, < header >, y < nav >.

Las etiquetas semánticas de HTML5

  • El < body > es donde se colocan todos los elementos que se van a mostrar y que forman el contenido de la página.
  • < div > es el tipo de caja genérica.
  • < main > es la caja que contiene todo lo principal de la página.
  • < header > es la caja destinada al encabezado de la página, aquí se incluye el logotipo y el nombre de la web.
  • La caja < footer > contiene la información que figura al pie de la página y es donde se ubica el ©, e información como el mapa de la página, autor, fecha y otros datos finales.
  • La caja < nav > contiene todos los enlaces de la página, ya sea a otras secciones de nuestro mismo sitio web o a páginas externas.
  • En las cajas < section > está el contenido genérico de la página.
  • Las cajas < article > también están destinadas a contener el contenido de la página, aunque normalmente suelen utilizarse para mostrar información más reducida y concreta y que puede ser independiente del resto.
  • Dentro de las cajas de tipo < aside > se ubican los contenidos que no tienen relación directa.

Estilos CSS y sus propiedades más utilizadas

Los estilos CSS son un conjunto de normas que indican cómo el navegador debe visualizar cada uno de los diferentes elementos que contiene una página web. Aunque estas normas habitualmente están relacionadas con aspectos visuales y de posicionamiento y colocación, con CSS3 cada vez existen más posibilidades (animaciones, transiciones, rotaciones...).

Una de las utilidades más evidentes de CSS es la posibilidad de aplicar un estilo (un conjunto de características) a múltiples páginas HTML, no teniendo que repetir los mismos estilos cada vez que creamos un HTML nuevo. En este caso se podría utilizar un único fichero CSS externo (con formato .css) que afectaría al mismo tiempo a múltiples páginas HTML, o bien incluir el código CSS dentro del mismo fichero HTML (dentro del < head >), o utilizar estas dos posibilidades conjuntamente.

Selectores

Los selectores CSS son una lista de uno o más elementos HTML (nombrándolos a través de su id, class o tipo de etiqueta) que indican a quien van a afectar las sentencias CSS que siguen al selector.

Propiedades

Que afectan al texto:

  • color chocolate; Cambia el color del texto.
  • font-family: helvetica; Cambia la tipografía.
  • font-size: 20px; Cambia el tamaño de la letra.
  • font-weight: bold; Muestra el texto en negrita.
  • text-shadow: 2px 4px 6px #1162ac; Define una sombre en el texto.
  • text-align: center; Alineación del texto.
  • line-height: 70px; Alineación vertical del texto.

Que afectan a las cajas

  • background-color: olive; Define un color de fondo de una caja.
  • padding: 20px;Define el espacio existente entre el contenido y el borde de la caja.
  • margin: 35px;Define el espacio existente entre cajas o bien entre cajas y los márgenes de la página.
  • box-shadow: 2px 7px 5px gray;Para definir una sombra de una caja.
  • float:left o right; Alinear a la derecha o la izquierda.

Que afectan a los enlaces:

  • text-decoration: none; Elimina el subrayado que aparece por defecto en cualquier enlace.
  • color: chocolate; Para definir el color que debe tener el enlace cuando el cursor pase por encima es necesario utilizar el evento :hover, detrás del selector a.
  • cursor: default; Es posible definir un tipo de cursor (dentro de los existentes). Aunque los más habituales son "pointer" (el que aparece por defecto al crear un enlace).
